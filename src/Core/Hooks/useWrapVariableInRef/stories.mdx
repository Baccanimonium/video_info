#### Примеры подключения

```
import React, { Component } from "react"

import useStoreVarInRef from "@/Core/Hooks/useWrapVariableInRef"
const Controller = () => {
    coconst [startCord, setStartCord] = useState()
    const startCordRef = useStoreVarInRef(startCord)
    const handler = useCallback((nextCord) => setStartCord(nextCord - startCordRef), [])
...
}
```


Позволяет завернуть переменную в ref в одну строчку, просто улучшение жизни
