# Добавление декоратора проверки прав перед выводом кнопки
<Meta
    title="Core/Decorators/withCheckPermissionAvailabilityBeforeRender"
/>

#### Примеры подключения

```
import withCheckPermissionAvailabilityBeforeRender from "@/Core/Decorators/withCheckPermissionAvailabilityBeforeRender"
import BsButton from "@/Components/BsButton"

export default withCheckPermissionAvailabilityBeforeRender(BsButton)

```

или
```
import { DefaultPermissionAvailabilityChecker } from "@/Core/Decorators/withCheckPermissionAvailabilityBeforeRender"
import styled from "styled-components"
const MenuItem = styled(DefaultPermissionAvailabilityChecker)`
    color: red;
`
```


Требует контекст прав объекта системы


#### Пропсы декоратора

| Название | обязательна | Описание |
|---|---|---|
| `permissionKey` | + | Тип права один из 7 типов |
| `value`  | - | значение формы |

#### Используется

* В панели инструментов таблицы
* На страницах Списков для создания/удаления/редактирования сущностей
