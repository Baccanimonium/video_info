# Добавление декоратора инкапсуляции состояния страницы
<Meta title="Core/Decorators/withApplyAbleForm" />

#### Примеры подключения

```
import withApplyAbleForm from "@/Core/Decorators/withApplyAbleForm"
class Form {
...
}

export default withApplyAbleForm(Form)

```
Работает внутри с копией данных страницы.
Используется для того, чтобы вносить изменения в данные страницы и применять их разово по завершению редактирования


#### Пропсы декоратора

| Название | обязательна | Описание |
|---|---|---|
| `onSubmit` | + | Обработчик сохранению формы, будет вызван если форма валидна |
| `value`  | - | значение формы |

#### Используется

* В форме редактирование фильтров
* В Сделке при редактировании комментариев
* В правах при добавлении объектов системы
