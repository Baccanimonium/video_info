# Добавление декоратора загрузчика файлов
<Meta
    title="Core/Decorators/withFileInput"

#### Примеры подключения

```
import withFileInput from "@/Core/Decorators/withFileInput"
class FileInput {
    ...
}

export default withFileInput(FileInput)

```


Выступает буфером загрузки файлов
Перед сохранением файла в состояние страницы необходимо загрузить файл на сервер и получить его ***KEY***



#### Пропсы декоратора

| Название | обязательна | Описание |
|---|---|---|
| `onInput` | + | Обработчик ввода файлов, передается значение ответа от сервера |
| `allowedTypes`  | - | список разрешенных типов файлов |
| `value`  | - | значение поля ввода |
| `disabled`  | - | разрешен ли ввод |
| `id`  | - | ключ поля в данных выступает HTML id аттрибутом |
| `multiple`  | - | разрешен ли множественный ввод |
| `folderPath`  | - | путь сохранения файла - подставляется в запрос при отправке файла на сервер |

#### Используется

* В компоненте аватара
* В файловых инпутах на страницах
