# Контейнер drag & drop элементов
<Meta title="Core/RenderProps/WithDragItem" />

#### Назначение

Стандартная обработка **drag & drop**, поддержка флага указывающего на то что элемент активен

#### Подключение

```
import WithDragItem from "@/Core/RenderProps/WithDragItem"

<WithDragItem
  onDragStart={onDragStart}
  onDragEnd={onDragEnd}
>
    {({ isItemDragging, onDragStart, onDragEnd }) => (
        <NamingFieldContainer
          className="display-flex a-i-center"
          draggable={true}
          onDragStart={onDragStart}
          onDragEnd={onDragEnd}
          isItemDragging={isItemDragging}
        />
    )}
</WithDragItem>
```

onDragStart => функция, которая ожидает JSON на выходе который необходимо передать цели **drop**-а

#### Аргументы

| Название | обязательна | Описание |
|---|---|---|
| `children` | + | функция в которую передается перехватчик события клика |
| `onDragStart` | + | функция обработчик начала **drag** действия, должна возвращать данные необходимые цели |
| `onDragEnd`  | + | функция обработчик конца **drag** действия |




#### Используется в компоненте боковых фильтров по всему приложению

* главные страницы разделов
* таблица
* права
