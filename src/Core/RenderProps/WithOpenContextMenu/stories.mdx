# Контейнер позволяющий открыть контекстное меню
<Meta title="Core/RenderProps/WithOpenContextMenu" />

#### Назначение

Позволяет открывать контекстное меню
#### Подключение

```
import WithCloseWindow from "@/Core/RenderProps/withCloseWindow"

<WithOpenContextMenu
  onOpenContextMenu={openButtonsMenu}
  settings={contextMenuSettings}
>
  {(onOpenContextMenu) => (
    <OpenContextMenuButton
      type="button"
      disabled={dropDownButtons.some(({ disabled }) => disabled)}
      open={isContextMenuOpen}
      className={`m-l-5 btn  black-btn display-flex a-i-center j-c-center ${isContextMenuOpen ? "active" : ""}`}
      onClick={onOpenContextMenu}
    >
      <ExpandIcon className="pos-absolute no-pointer-events" />
    </OpenContextMenuButton>
  )}
</WithOpenContextMenu>
```

[Полный гайд по открытию контекстного меню](?path=/story/guides-открытие-контекстного-меню--page)

onOpenContextMenu => функция, которая ожидает получить объект конфигурации контекстного меню

#### Аргументы

| Название | обязательна | Описание |
|---|---|---|
| `children` | + | функция в которую передается перехватчик события клика |
| `settings` | - | конфиг контекстного меню |
| `onOpenContextMenu`  | + | функция обработчик события открытия контекстного меню |
| `onUpdateContextMenu`  | + | функция обработчик события обновления контекстного меню  |
| `fieldRef`  | + | **ref** поля вокруг которого открывается контекстное меню |
