# Персональный ресайзер для детей
<Meta title="Core/RenderProps/RenderResizer" />

#### Назначение

Нужен для изменения размера детей и хранение значений в **localStorage**

#### Подключение

```
import RenderResizer from "@/Core/RenderProps/RenderResizer"

<RenderResizer id={id}>
    {({ refContainer, onStartResize, getElementSize }) => {
        const wrapChildren = childrenWrapper(onStartResize, getElementSize)
        return (
          <StylePerfectScrollbar ref={refContainer}>
            {children.map(wrapChildren)}
          </StylePerfectScrollbar>
        )
    }}
</RenderResizer>
```

#### Аргументы

| Название | обязательна | Описание |
|---|---|---|
| `refContainer` | + | **ref** На родительский элемент детей |
| `onStartResize` | + | функция обработчик начала изменения размеров элемента |
| `children`  | + | функция потомок в которую передается отпут контроллера |
| `initialCord`  | - | значение на которое нужно увеличивать координату |
| `minSize`  | - | минимальный размер элемента |
| `defaultSize`  | - | размер элемента по умолчанию |
| `pixelsAutoScrolled`  | - | интеграция со скроллом, сколько пикселей было проскроллено при ресайзе |
| `id`  | - | ключ окна в localstorage |

#### Используется в компоненте боковых фильтров по всему приложению

* главные страницы разделов
* таблица
* права
