# Контейнер для перетасовки элементов
<Meta title="Core/RenderProps/WithMoveAbleItems" />

#### Назначение

Позволяет менять элементы местами или удалять их

#### Подключение

```
import WithMoveAbleItems from "@/Core/RenderProps/WithMoveAbleItems"

<WithDragItem
  onDragStart={onDragStart}
  onDragEnd={onDragEnd}
>
    {({ isItemDragging, onDragStart, onDragEnd }) => (
        <NamingFieldContainer
          className="display-flex a-i-center"
          draggable={true}
          onDragStart={onDragStart}
          onDragEnd={onDragEnd}
          isItemDragging={isItemDragging}
        />
    )}
</WithDragItem>
```

onDragStart => функция, которая ожидает JSON на выходе который необходимо передать цели **drop**-а

#### Аргументы

| Название | обязательна | Описание |
|---|---|---|
| `children` | + | функция в которую передается перехватчик события клика |
| `onDragStart` | + | функция обработчик начала **drag** действия, должна возвращать данные необходимые цели |
| `onDragEnd`  | + | функция обработчик конца **drag** действия |

#### output контроллера

| Название | Описание |
|---|---|
| `isItemMoving`| Флаг указывающий что какой-то элемент выбран пользователем |
| `value`| Валуе которое должно быть выведено в **UI** |
| `onDragStart`| Функция обработчик начала **drag** события |
| `onDragEnd`| Функция обработчик конца **drag** события |
| `onDragEnter`| Функция обработчик **drag** события на элементе отличном от активного  |
| `onDragOverDeleteContainer`| Функция заглушка необходимая для работы контейнера удаления элемента |
| `onDropAndDelete` | функция обработчик конца **drop** события, нужна для удаления элемента из списка |




#### Используется в компоненте боковых фильтров по всему приложению

* главные страницы разделов
* таблица
* права
