# Добавление декоратора добавляющего контекст и кнопку сохранения значения поля ввода
<Meta
    title="Core/Decorators/WithSubmitButtonHOC"
    component={WithSubmitButtonHOC()}
/>

#### Примеры подключения

```
export const fieldConfig = [
    {
        id: "COMENT",
        component: WithSubmitButtonHOC(BsInput, SubmitButton)
    }
]
export default WithSubmitButtonHOC(Page)

```

Отличие от WithSubmitContainerHOC только в отображении
#### Пропсы

| Название | обязательна | Описание |
|---|---|---|
| `ButtonLabel` | - | Компонент содержимого кнопки сохранения(иконка) |
| `value` | - | значение поля ввода |


#### Используется

* В формах, где редактирование значения происходит в контекстном меню по клику на кнопку
